# ⬜Whitepace — составляйте планы быстрее и легче

## 📖 О проекте
Whitepace - это программное обеспечение для управления проектами, позволяющее вашим командам сотрудничать, планировать, анализировать повседневные задачи и управлять ими

### 🎯 Ключевые особенности:
* Работа в команде: Делитесь проэктами с другими и работайте с ними вместе.
* Обширный редактор: Поддержка фото, видео, pdf и аудио. .
* Поддержка ваших браузеров: Идеально работает на Firefox, Microsoft Edge, Chrome, Yandex и Opera.
* Работайте где угодно и на чем угодно: Whitepace доступен на андроид и IOS, Windows, MacOS и Linux
